Основное свойство, для объектов, способные говорить в чате:
	person		[ник_на_английском,цвет_ника_в_hex,цвет_сообщения_в_hex]

[bonus_health]
Слой, пересечение с которым даёт hp.

[bonus_speed]
Слой, пересечение с которым даёт скорость.

[bonus_damage]
Слой, пересечение с которым даёт повышенный урон.

[bonus_god]
Слой, пересечение с которым даёт бессмертие.

[bonus_super_weapon]
Слой, пересечение с которым делает тройную стрельбу.

[bonus_weapon_homing]
Слой, пересечение с которым делает автонаводку стрельбы.

[bot]
Объект врага
	action - move (ходить), stop (Стоять)
	direction - Направление движения
	type - friend (Друг), enemy (враг)
	
[player]
Объект игрока 

[movingPlatform]
Объект передвигающийся платформы 
	Свойства:
	direction: движение
	
[pointer]
Объект указания передвижения передвигающийся платформы
	Свойства:
	direction: движение
	
[stair]
Объект лестницы 

[collision]
Слой столкновений

[collision_up]
Слой в котором столкновение только по полу, также все физические объекты могут спрыгнуть с него

[falling]
Слой столкновений, но через X сек пропадает тайл

[light_block]
[ВЫКЛЮЧЕНО] Слой блокировки света, использовать инструмент - точки

[light]
Объект света 
	Свойства:
	angle: угол поворота
	color: [r, g, b] - цвет
	scale: [x, y] - растягивание
	type: cone или point
	

[damage_all]
Слой, который убивает сразу

[damage]
Слой, который наносит урон и толкает игрока

[damage_box]
Слой столкновений, но может быть повреждён пулей

[scripts]
Объекты этого слоя по имени запускают скрипты

[collision_angle_left]
Объект, где столкновение проходит по углу, верхушка находится слева

[collision_angle_right]
Объект, где столкновение проходит по углу, верхушка находится справа

[teleport]
Телепорт
	Свойства:
	to - Название объекта на карте, на который следует телепортироваться (В поле на карте установить draw no)
